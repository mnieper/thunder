Thunder

Thunder is a virtual machine based on GNU lightning. The virtual machine is intended to be used as a compilation target of dynamic programming languages like Scheme and JavaScript.

Downloading

Releases for Thunder can be found via HTTP.

Documentation

Documentation for Thunder is available online. You may also find more information about Thunder by running info thunder or man thunder. A brief summary is available by running thunder --help.
